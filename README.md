
#  Heart Disease Prediction Analysis

## 📌 Overview

This project focuses on Exploratory Data Analysis (EDA) of a Heart Disease Prediction dataset containing medical information from patients.

The analysis helps understand:

* Important health indicators related to heart disease
* Relationships between clinical features
* Data distribution and correlations
* Patterns that may help in future predictive modeling

The notebook uses Python data analysis and visualization libraries to explore the dataset in a clean and beginner-friendly way.

---

## 🎯 Project Objectives

* Analyze patient medical records
* Explore feature distributions and trends
* Identify relationships between clinical variables
* Visualize correlations between features
* Build a strong foundation for future Machine Learning models

---

## 📊 Dataset Features

The dataset includes important medical attributes such as:

* Age
* Blood Pressure (BP)
* Cholesterol
* Chest Pain Type
* ECG Results
* Maximum Heart Rate
* Thallium Test Results
* Other clinical measurements
